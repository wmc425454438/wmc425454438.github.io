Javascript Template Mode

テンプレートメソッドパターンは、パッケージの変更によりシステムのスケーラビリティを向上させる典型的な設計パターンです。オブジェクト指向言語では、テンプレートメソッドパターンを使用するプログラムでは、サブクラスのメソッドタイプと実行順序は変わらないため、ロジックのこの部分は、親クラスのテンプレートメソッドに抽象化されます。サブクラスメソッドの実装方法は変更可能なため、これを採用します部分的に変更されたロジックはサブクラスにカプセル化されます。新しいサブクラスを追加することにより、システムに新しい関数を変更せずに追加できます。抽象親クラスと他のサブクラスを移動することも、開放性と閉鎖性の原則に沿っています。
しかし、JavaScriptでは、テンプレートメソッドパターン、高階関数を実装するために同じパターンに従う必要はありません。より良い選択です。
